# Javed Ansari Portfolio

A fast, static React portfolio built with Vite. It includes sections for summary, education, experience, projects, certifications, skills, and contact details.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
